# Amna Riz Shoe Company — Shoe Brand Landing Page

A single-page shoe brand website built with vanilla HTML, CSS, and JavaScript.

**Live demo → [consult-with-riz.github.io/shoe-brand/shoe-brand.html](https://consult-with-riz.github.io/shoe-brand/shoe-brand.html)**

## Features

- Custom animated cursor with lag ring
- Fullscreen hero with floating product animation
- Auto-scrolling marquee ticker
- Product grid with hover effects
- Scroll-reveal animations
- Fully responsive

## Stack

Pure HTML / CSS / JS — no frameworks, no dependencies.

## Usage

```bash
git clone https://github.com/consult-with-riz/shoe-brand.git
```

Then open `shoe-brand.html` in a browser. That's it.
